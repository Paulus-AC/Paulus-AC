# 💫 About Me:
I’m a backend and network professional with experience building RESTful APIs and backend services using Laravel, PHP, and SQL. My work has included mail service integrations, web scraping, and data migrations, where I focused on writing clean and efficient solutions.<br><br>Recently, I became a Cisco Certified Network Associate (CCNA), and I’m expanding my career into network engineering. I’m passionate about routing, switching, and network security, and I enjoy combining my software background with networking to solve real-world problems.<br><br>Outside of tech, I enjoy offline open-world games, basketball, and continuous learning.
